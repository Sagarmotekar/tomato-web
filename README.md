# 🍔 Food Del - Food Delivery Web Application

Food Del is a full-stack food delivery web application developed using the MERN stack. The application allows users to browse food items, manage their cart, place orders, and view their order history through a modern and responsive interface.

## 🚀 Features

* User Authentication
* Food Menu Browsing
* Add to Cart Functionality
* Cart Management
* Place Orders
* Order Verification
* My Orders Page
* Responsive Design
* Secure User Sessions

## 🛠️ Tech Stack

### Frontend

* React.js
* Vite
* React Router DOM
* CSS3

### Backend

* Node.js
* Express.js

### Database

* MongoDB

### Authentication

* JWT Authentication

## 📂 Modules

* Home
* Cart
* Place Order
* Login & Registration
* Order Verification
* My Orders

## 📱 Responsive Design

The application is optimized for desktop, tablet, and mobile devices.
